# 3D_Pallet_Loading
3 different strategies for soloving 3d pallet loading problem
The decoder is designed based on 'Extreme point' proposed by Crainic et al., 2008, "Extreme Point-Based Heuristics for Three-Dimensional Bin Packing
The boxes and pallets datas are from Centraal Boekhuis 'https://www.cb.nl/'
Use of generative AI tools: 1.for debugging propose  2.for adding ProcessPoolExecutor
